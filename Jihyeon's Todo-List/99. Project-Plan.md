## 1. 프로젝트 개요 

### 프로젝트 이름
My Todo List

### 한 줄 설명
할 일을 추가하고 완료 체크할 수 있는 간단한 투두 리스트 앱

### 만들게 된 이유 
CRUD(생성, 조회, 수정, 삭제)의 기본 개념을 익히고 싶었고,
나중에 Java Spring Boot 벡엔드와 연동하는 연습을 하기 위해 선택하였습니다.
실생활에서도 쓸 수 있어서 동기부여가 될 것 같습니다.

---
## 2. MVP 범위 설정 (MoSCoW)

### ✅ Must (반드시 구현) - 최대 3개

1. **할 일 추가**: 텍스트 입력 후 버튼 클릭으로 목록에 추가 
2. **할 일 삭제**: 삭제 버튼 클릭으로 목록에서 제거 
3. **완료/미완료 토글**: 체크박스 클릭으로 상태 변경
